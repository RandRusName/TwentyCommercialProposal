# Known Limitations

- Remote metadata plan/apply was not executed because no API key was available.
- Runtime GraphQL operation names for custom object records are generated by
  Twenty after metadata sync. The adapter isolates this in
  `src/services/twenty-record-repository.ts`.
- The command menu item is scoped to Opportunity object context, but final
  single-record validation is done in the front component and logic function.
- No Company command entry point is implemented.
- No DOCX/PDF generation is implemented.
- No document-service integration is implemented.
- Files storage is modeled but not used yet.
- No email/send workflow is implemented.
- No row-level production permission hardening has been performed.
